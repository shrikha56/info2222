## Project Overview
This project is a hi-fi prototype for a collaborative group-work platform designed for student teams. The system combines task management, messaging, a Kanban board, a leaderboard, project and document browsing, a call interface, and collaborative document editing in one integrated platform.

The hi-fi prototype was developed in Next.js using TypeScript and Tailwind CSS, based on our paper prototype designs.

## Tech Stack
- Next.js
- TypeScript
- React
- Tailwind CSS
